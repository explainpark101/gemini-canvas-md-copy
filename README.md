# gemini-canvas-md-copy

To install dependencies:

```bash
bun install
```

To build:

```bash
bun pack
```

# Added Features
- KaTeX / LaTeX support (math-inline, math-block) — `data-math` → `$...$` / `$$...$$`
- **Context menu**: 텍스트 선택 후 우클릭 → "Copy selection as Markdown"으로 선택 영역 HTML을 Markdown으로 변환하여 복사
