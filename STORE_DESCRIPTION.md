# Chrome Web Store 확장 프로그램 설명

Gemini에서 Canvas를 통해 긴 문서나 코드를 생성하면, 그대로 복사해서 다른 곳에 붙여넣기가 쉽지 않습니다. Gemini Canvas Markdown Copy는 이 문제를 해결합니다.

## 주요 기능

### 1. 원클릭 마크다운 복사
- Gemini Canvas 응답 영역이 보일 때만 플로팅 버튼이 나타납니다.
- 버튼을 누르면 HTML이 마크다운(# 제목, 굵게, [링크](url))으로 변환되어 클립보드에 복사됩니다.
- 노션, GitHub, 블로그, 문서 등 마크다운을 지원하는 모든 곳에 그대로 붙여넣을 수 있습니다.

### 2. 고급 마크다운 변환
- 플로팅 버튼을 클릭하여 마크다운으로 변환된 내용을 바로 복사할 수 있습니다.
  - 복사된 다음에는 "Copied" 로 일시적으로 변경됩니다.
- 표, 리스트, 코드 블록 등 다양한 요소를 지원합니다.
- Gemini의 답변을 드래그하여 선택하고, 선택된 부분에 우클릭을 누른 뒤, 우클릭 메뉴에서 `Copy selection as markdown` 항목을 선택하면, 선택된 부분이 마크다운 형식으로 클립보드에 복사됩니다.

### 3. 사용하기 편한 UI
- 버튼을 꾸욱 눌러 드래그해서 원하는 곳에 둘 수 있습니다.
- 위치는 자동 저장되어 다음 방문 시에도 그대로 유지됩니다.
- 복사할 markdown이 없을 때는 버튼이 자동으로 숨겨져 화면을 가리지 않습니다.

## 이런 분들께 추천합니다

- Gemini로 작성한 문서를 노션, 블로그, GitHub에 옮기시는 분
- 마크다운 형식으로 정리된 텍스트가 필요하신 분
- Gemini Canvas 응답을 수동으로 복사·정리하는 데 시간을 쓰시는 분
